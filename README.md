
# Soccer Player Stats Analyzer

Exploratory data analysis of European soccer players and match appearances using Python and Pandas.

## Dataset
Data sourced from [Kaggle - Player Scores](https://www.kaggle.com/datasets/davidcariboo/player-scores), containing 44,000+ players and 1.8M+ match appearances across top European leagues.

## Analysis
- Top 20 all-time goal scorers
- Top 15 countries by number of players
- Top 20 all-time assisters
- Top 15 clubs by total goals scored

## Tools
Python, Pandas, Matplotlib, Jupyter Notebook
